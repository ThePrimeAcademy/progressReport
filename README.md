# ProgressReport Generator
Grabs data from classmarker through their API and returns the data in a presentable report card format

## 🌟 Highlights
* **Date Range:** Allows to set date range as well as certain day of the week.
* **Export to PDF** Export the report card to PDF.
* **Email (In Progress):** Can set the email of the parents of students and send the report card all at once.
